Second project of the Full Stack Web Development career in [Ac√°mica](https://www.acamica.com/desarrollo-web-full-stack).

In this project i built a front-end webpage, using HTML, CSS and JavaScript without any framework. The site is based on an UX design.

All GIFs are from the Giphy [API](https://developers.giphy.com/).

### Features:
- Recommended section.
- Trending section.
- GIF recording.
- GIF upload to Giphy.
- GIF search.
- Desktop-only responsive
- Light/dark theme.
